## requirements
- [Python 3.9+](https://www.python.org/)
- [PY5 module](https://py5coding.org/index.html)
  - and Java 17+
## How to Start

1. Install and correctly set up the required configuration
  - Python
  - PY5
  - `JAVA_HOME`
2. Run `python template.py`
